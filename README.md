## ✨ Возможности

### Основные функции
- ✅ **Добавление транзакций** — доходы и расходы с категориями
- ✅ **Просмотр баланса** — общая статистика с визуализацией
- ✅ **История операций** — таблица с фильтрацией по типу и категории
- ✅ **Умный поиск** — по описанию, категории, сумме, дате
- ✅ **Удаление транзакций** — простая коррекция данных

### Аналитика и планирование
- 📊 **Статистика по месяцам** — доходы/расходы/баланс за каждый месяц
- 🎯 **Управление бюджетами** — установка лимитов по категориям
- 📈 **Визуализация** — круговые и столбчатые диаграммы расходов
- 📅 **Умный ввод даты** — быстрый выбор: сегодня, вчера, на этой неделе

### Интерфейс
- 🎨 **Цветной интерфейс** — понятное разделение информации
- 📊 **Красивые таблицы** — форматированный вывод данных
- 📊 **Прогресс-бары** — визуальное отслеживание бюджетов
- 💾 **Автосохранение** — данные сохраняются автоматически

## 🚀 Установка

### Вариант 1: Установка через Python (рекомендуется)

#### 1. Установите Python
- Скачайте Python 3.7 или выше с [официального сайта](https://www.python.org/downloads/)
- При установке **обязательно отметьте** галочку "Add Python to PATH"

#### 2. Скачайте приложение
```bash
git clone https://github.com/ваш-username/finance-app.git
cd finance-app

Или просто скачайте файл finance_app.py вручную.

3. Установите зависимости
bash
pip install -r requirements.txt
Или установите библиотеки по отдельности:

bash
pip install colorama tabulate matplotlib
4. Запустите приложение
bash
python finance_app.py

Вариант 2: Запуск без установки Python (Windows)
Скачайте готовый .exe файл из раздела Releases

Распакуйте архив

Запустите finance_app.exe

Вариант 3: Установка в виртуальное окружение (для разработчиков)
bash
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Mac/Linux)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python finance_app.py

Посмотрите баланс — выберите пункт 2

Установите бюджет — выберите пункт 7 и установите лимиты

Проанализируйте статистику — выберите пункт 5




📖 Инструкция по использованию
1️⃣ Добавление транзакции
Выберите тип: 1 - доход, 2 - расход

Выберите категорию из списка

Введите сумму (только положительные числа)

Введите описание (необязательно)

Выберите дату:

д - текущая дата

н - выбрать из предложенных вариантов:

Сегодня

Вчера

На этой неделе

В этом месяце

Другая дата (формат: ДД.ММ.ГГГГ)

2️⃣ Просмотр баланса
Показывает:

Общую сумму доходов

Общую сумму расходов

Текущий баланс

Расходы по категориям с процентами

Прогресс выполнения бюджетов

3️⃣ История транзакций
Возможности:

Фильтрация по типу (доходы/расходы)

Фильтрация по категории

Сортировка по дате (новые сверху)

Итоговая сумма по отфильтрованным транзакциям

4️⃣ Поиск транзакций
Поиск осуществляется по:

Описанию — частичное совпадение текста

Категории — все транзакции выбранной категории

Сумме — больше или меньше указанного значения

Дате — точное совпадение даты

Типу — все доходы или все расходы

5️⃣ Статистика и графики
Статистика по месяцам — таблица доходов/расходов/баланса

Круговая диаграмма — распределение расходов по категориям

Столбчатая диаграмма — сравнение сумм по категориям

Управление бюджетами — установка и отслеживание лимитов

6️⃣ Управление бюджетами
Просмотреть бюджеты — увидеть текущие лимиты и прогресс

Установить бюджет — выбрать категорию и ввести лимит

Удалить бюджет — убрать лимит для категории

7️⃣ Удаление транзакций
Показываются последние 10 транзакций

Выберите номер транзакции для удаления

Подтверждение не требуется (можно отменить вводом 0)

8️⃣ Сохранение данных
Данные сохраняются автоматически после каждой операции

Можно сохранить вручную через пункт 8

Файл данных: finance_data.json (создается автоматически)

9️⃣ Выход
Автоматическое сохранение перед выходом

⚠️ Устранение проблем
Ошибка: ModuleNotFoundError: No module named 'colorama'
Решение: Установите недостающие библиотеки:

bash
pip install colorama tabulate matplotlib
Ошибка: pip не является внутренней командой
Решение:

Убедитесь, что Python установлен и добавлен в PATH

Используйте python -m pip install вместо pip install

Ошибка: 'utf-8' codec can't decode byte
Решение:

Убедитесь, что файл finance_data.json сохранен в кодировке UTF-8

Удалите старый файл данных (создастся новый)

Графики не отображаются
Решение:

Убедитесь, что matplotlib установлен

Закройте другие окна с графиками

Попробуйте запустить программу в обычной консоли (не в IDLE)

Русские буквы отображаются некорректно
Решение (Windows):

bash
chcp 65001  # Переключение на UTF-8 в консоли
Или запустите программу в PowerShell или Windows Terminal.
